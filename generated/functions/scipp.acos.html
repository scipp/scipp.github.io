scipp.acos#

scipp.acos(x, *, out=None)#

Element-wise inverse cosine.

Parameters:

  • x (scipp.typing.VariableLike) Input data.

  • out (Optional[TypeVar(VariableLikeType, Variable, DataArray, Dataset, DataGroup[Any])], default: None) Optional output buffer.

Returns:

Same type as input The inverse cosine values of the input in radians.
